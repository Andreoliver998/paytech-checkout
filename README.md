# PayTech Checkout

Node.js checkout system for PayTech payment processing using Mercado Pago.

## Features

- Secure payment processing
- Mercado Pago integration
- Production-ready deployment
- Node.js backend
- PM2 process management

## Installation

```bash
git clone https://github.com/Andreoliver998/paytech-checkout.git
cd paytech-checkout
npm install
```

## Run Locally

```bash
npm start
```

## Production

```bash
pm2 start ecosystem.config.js
```

## Production URL

```text
https://checkout.paytech.app.br/pay
```
